# ชุดเกมฝึกคลิกเมาส์ ป.1

ไฟล์นี้เป็นชุดเกม HTML สำหรับฝึกนักเรียนใช้เมาส์

## รายชื่อเกม
- คลิกลูกโป่ง
- จับดาว
- จับปลา
- เก็บชีสให้หนู
- ไฟเขียวไฟแดง

## วิธีใช้งาน
1. เปิดไฟล์ `index.html` ด้วย Google Chrome หรือ Microsoft Edge
2. เลือกเกม
3. กดเริ่มเล่น

## วิธีอัปขึ้น GitHub Pages
1. สร้าง Repository ใหม่ใน GitHub
2. อัปโหลดไฟล์ทั้งหมด ได้แก่ `index.html`, `style.css`, `script.js`, `README.md`
3. ไปที่ Settings > Pages
4. เลือก Deploy from a branch
5. เลือก branch `main` และ folder `/root`
6. กด Save
