# ชุดเกมฝึกคลิกเมาส์ ป.1 เวอร์ชัน 3

## แก้ไขสำคัญ
เกมไฟเขียวไฟแดงไม่ค้างแล้ว

- ไฟเขียว: ต้องคลิกเพื่อได้คะแนน
- ไฟแดง: ห้ามคลิก รอเฉย ๆ แล้วเกมไปต่อเอง
- ไฟเหลือง: ห้ามคลิก รอเฉย ๆ แล้วเกมไปต่อเอง
